# Projeto Final
    Projeto Final referente a cadeira de Programação Orientada a Objetos.
    Temática: Banco
    Resumo: Sistema referente ao funcionamento de um banco.
            Correntistas (user) - Pessoa Fisica ou Pessoa Juridica.
            Contas - Conta Corrente Comum       (apenas debito).
                     Conta Corrente Com Limite  (debito+credito).
                     Conta Poupança             (apenas debito).

    Opções da main:
            Gerente - Controla a criação e manipulação das contas.
            Correntista - Controla os dados da sua conta.
